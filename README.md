# Daphnia Clinics Complex

A modern web-based management system for organizing guard schedules and staff duty rotations at the Daphnia Clinics Complex.

---

## Overview

Daphnia Clinics Complex is designed to simplify security guard scheduling through an intuitive and responsive web interface.

The application provides secure login, monthly calendar management, guard information management, duty tracking, and shift replacement handling.

---

## Features

- Secure Login System
- Password Management
- Monthly Duty Calendar
- Guard Information Management
- Shift Rotation
- Shift Cancellation
- Substitute Guard Assignment
- Duty Summary Reports
- Mobile Friendly Design
- Local Storage Support
- Responsive Interface

---
